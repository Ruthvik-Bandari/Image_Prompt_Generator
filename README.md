# AI Image Analysis & Prompt Generator

A full-stack application that leverages OpenAI's GPT-4 Vision API to analyze images and generate optimized prompts for AI image generation tools.

## Features
- Upload and analyze any image using AI
- Generate multiple prompt formats (JSON, Cartoon, Cinematic, Detailed)
- Modern React UI with marble black and blue theme
- RESTful API with Spring Boot
- One-click copy functionality

## Tech Stack
- **Backend**: Java 17, Spring Boot 3.1.5, Maven
- **Frontend**: React 18.2, JavaScript ES6+
- **AI**: OpenAI GPT-4 Vision API

## Quick Start

### Prerequisites
- Java 17+
- Node.js 16+
- Maven 3.6+
- OpenAI API Key

### Installation
1. Clone the repository
2. Copy `application.properties.example` to `application.properties` and add your OpenAI API key
3. Run backend: `cd backend && mvn spring-boot:run`
4. Run frontend: `cd frontend && npm install && npm start`
5. Open http://localhost:3000
